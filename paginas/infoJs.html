¿Que es JavaScript?

Es un lenguaje de programación ampliamente utilizado que se utiliza principalmente en el desarrollo web. Es un lenguaje interpretado de alto nivel que se ejecuta en el navegador del usuario, lo que permite crear interactividad en páginas web dinámicas. JavaScript es fundamental para muchas funcionalidades modernas en la web, como la validación de formularios, efectos visuales, animaciones, comunicación asíncrona con el servidor (AJAX), y más recientemente, el desarrollo de aplicaciones web completas utilizando tecnologías como Node.js.

Adivina el numero del 1 al 100

¿Como funciona?

  • 1. Generación del número aleatorio: Math.floor(Math.random() * 100) + 1: El juego genera este número de manera aleatoria. Este número es lo que tienes que adivinar.

  • 2. Adivinando el número: Ahora, te piden que ingreses un numero en un campo de texto. Este numero que ingresas es tu intento para adivinar el numero secreto.

  • 3. Evaluación de tu intento: Después de ingresar tu número: Si tu número está fuera del rango válido (menor que 1 o mayor que 100), te indican que debes ingresar un número válido. Si el número que ingresaste es válido, el juego contará tus intentos. Compara tu número con el número secreto: Si adivinas el número exactamente, te felicitan y te dicen en cuántos intentos lo lograste. Si tu número es menor que el número secreto, te dicen que intentes con un número más grande. Si tu número es mayor que el número secreto, te sugieren intentar con un número más pequeño.

  • 4. Reiniciar el juego: Después de adivinar correctamente el número o cuando quieras comenzar de nuevo, el juego genera un nuevo número aleatorio para que empieces a adivinar de nuevo. También reinicia el contador de intentos para registrar tu progreso desde cero.
